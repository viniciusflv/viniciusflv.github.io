# How to create diagrams <br /> and charts in Markdown?

## Mermaid lets you create diagrams and visualizations using text and code.

---

### Diagrams

#### Flowchart

##### Simple node

<Mermaid
  chart={`
    flowchart LR
      node
  `}
>

```mermaid
flowchart LR
  node
```

</Mermaid>

##### Square node

<Mermaid
  chart={`
    flowchart LR
      square[square node]
  `}
>

```mermaid
flowchart LR
  square[square node]
```

</Mermaid>
